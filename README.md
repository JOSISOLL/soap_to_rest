SOAP_TO_REST
=================
API with Flask to request a SOAP Service

Usage
-----

Clone the repo:

    git clone https://github.com/JOSISOLL/soap_to_rest.git
    cd soap_to_rest

Create virtualenv:

    virtualenv venv
    source venv/bin/activate
    pip install -r requirements.txt

Run the sample app

    python app.py

Test the endpoints.
